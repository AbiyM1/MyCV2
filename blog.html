Relative Sizing in CSS

If You are familiar with CSS, you already know what relative and absolute sizing of objects. Setting font size of an element is quite easy. This concept becomes fairly vague in the domain of responsiveness. There are different arguments for setting the size of an element. Px, em and rem to be exact. rem is used to give tuning control over the sizing of an element to end-users. By default, 1rem represents 16 pixels. This changes based on what the user set their font-size to in their browsers. This is a property shared with em sizing.
