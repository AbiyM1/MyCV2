CSS Units for font size

Setting font size of an element is quite easy. This concept becomes fairly vague in the domain of responsiveness and usability. There are different units for setting the size of an element. Px, em and rem to be exact. rem is used to give tuning control over the sizing of an element to end-users. By default, 1rem represents 16 pixels. This changes based on what the user set their font-size to in their browsers. This is a property shared with em sizing. CSS Units for font size

The default width of a body is 16px or 1em. The browser interprets this to be the default size unless the user manually sets the size for an element in the body. This means if the manually given size is greater than what's given to the body by default, the body will extend itself to encompass the element.

What's new in CSS3

CSS3 is a revamped version of css. In this version, many features were added and improved. For instance, box-sizing feature that is very significant for assigning a restricted space to a partition was added. Some other features of CSS3 include animation and transision property, advanced selectors, gradients, webfonts, 3d transforms, text shadow.
